# Automated Invoice Analytics

An End-to-End Data Analytics project that transforms raw invoice transaction data into meaningful business insights using **Python, MySQL, Microsoft Excel, and Microsoft Power BI**. The project demonstrates the complete analytics workflow, including data cleaning, SQL analysis, KPI generation, dashboard creation, and business reporting.

---

## Project Overview

Businesses generate thousands of invoices every day, making it difficult to manually monitor sales performance, customer behavior, and revenue trends. This project automates the analysis process by converting raw invoice data into interactive dashboards and actionable business insights.

---

## Objectives

- Clean and preprocess invoice data using Python.
- Perform business analysis using SQL.
- Generate key business KPIs.
- Build an Excel dashboard for reporting.
- Create an interactive Power BI dashboard.
- Support data-driven business decision-making.

---

## Dataset Information

- **Source:** Kaggle
- **Domain:** Retail / Sales Analytics
- **Format:** CSV
- **Records:** ~10,000 invoice transactions

## Project Workflow

1. Collected the invoice dataset from Kaggle.
2. Cleaned and transformed the raw data using Python.
3. Generated additional analytical columns.
4. Performed business analysis using SQL queries.
5. Created KPI reports and Pivot Dashboards in Excel.
6. Developed an interactive dashboard in Microsoft Power BI.
7. Documented the complete project with business insights.

## Key Performance Indicators (KPIs)

- Total Revenue

## SQL Analysis

The SQL analysis focused on answering business-oriented questions such as:

- Top revenue-generating cities
- Average revenue by customer occupation
- Product-wise quantity sold
- Revenue analysis using aggregate functions
- Business reporting using GROUP BY, HAVING, ORDER BY, and LIMIT
